list(target_object: str)[source]¶

Search Google storage target location and return an iterator. This iterator generates all files under the target location. If the target is a single file, the iterator only one object.

Parameters

target_object – Target Google storage location. This could be a file or a folder. This is a string that looks like “gs://xxxxx”

Returns

An iterator that iterates over the target location. Each item is a Blob object.
